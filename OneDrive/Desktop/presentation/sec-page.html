ID Selector

The choice for an id name can be anything, but it’s suggested to use simple, but meaningful names with no space. An Id must be unique, meaning that you cannot repeat the same id name. Some editors won’t even allow you to use the same Id name. You declare an Id name in your html file, inside the opening tag of the element you choose to add an attribute.
