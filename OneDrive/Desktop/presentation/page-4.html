ID Selector in JavaScript

In JavaScript you can manipulate an element by also using a specific id name. Given an id name in HTML, you can access it in JavaScript by using the method getElementById(), or querySelector(). Both of these methods will work, however, querySelector is more flexible than getElementById, it uses valid CSS selectors. To manipulate an id name using getElementById(), you use the syntax document.getElementById(), inside the parentheses between quotes (single/double) you declare the id name:
document.getElementById(“main-heading”).
To access an id name using querySelector(), you declare the type of selector follow by the name given in html, in this case we have an id and, an id is identify by the hash character:
document.querySelector(“#main-heading”).
