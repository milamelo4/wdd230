Class Selector in JavaScript

You can use JavaScript to manipulate an HTML element with a specify class name, using the method getElementByClassName(), or querySelector(). To access a class name in JS, the syntax is: document.getElementByClassName(), inside the parentheses between quotes(single/double) you declare the class name:
document.getElementByClassName(“main-heading”).
To access a class name using document.querySelector(), you declare the type of selector follow by the name given in html, in this case we have a class, which is identify by a dot (.) character: document.querySelector(“.main-heading”).
